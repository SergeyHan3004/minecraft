СТИВ

Стив - вымошленный песонаж из видеоигры Minecraft. Созданный шведским разработчиком видеоигр Маркусом "Нотчем" Перссоном и представленный в начальной Java-версии Minecraft, которая впервые была публично выпущена 17 мая 2009 года, Стив является одним из девяти скинов игрового персонажа по умолчанию, доступных для игроков современных версий Minecraft.
